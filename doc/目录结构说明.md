# 目录结构说明

## iCAX-Engine

Engine 是 C++ 后台框架文档区，目录与 `src/icax-engine` 的 Foundation、Framework 等层级对齐。

## iCAX-UI

UI 是 H5 前端公共框架文档区，目录与 `src/iCAX-UI` 的 UIContainer、ApplicationProxy、FrontendSDK 等层级对齐。

### Foundation

Engine 最底层基础能力的规格与方案文档，目录与 `src/icax-engine/foundation` 对齐。

### Backend概要文档

C++ 后台当前 ApplicationHost、ProductRuntime、Project、Mailbox、PDO、Resource 和注册表关系的总览。

#### Task

C++ Task 基础库文档，包含规格文档与方案文档。

## Apps

具体的业务应用目录，每个应用拥有一个目录。应用可以选择 Qt、WPF、H5 或其他前端宿主形态，但与 C++ 后台交互的基本方式仍是 Mailbox、PDO 和 Resource。

### backend

应用侧 C++ 扩展或宿主入口。

### frontend

应用侧 UI 代码或 UI 宿主代码。

### protocol

应用级 Mailbox/PDO 协议约定。
