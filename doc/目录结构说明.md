# 目录结构说明

## iCAX-Engine

Engine，修改频次极低

### Foundation

Engine 最底层基础能力的规格与方案文档，目录与 `src/icax/foundation` 对齐。

#### Task

C++ Task 基础库文档，包含规格文档与方案文档。

## iCAX-UIFoundation

UI基础库，基于iCAX-Engine封装UI基础部分，供后续可以基于此实现产品代码。此处基于iCAX-Engine提供的PDO、mailbox机制封装UI层与其通信。

修改频次低

### H5

基于WebView2提供的H5页面基础。

### WPF/WINFORM

提供WPF/WONFOPRM的页面基础。

### QT

提供QT页面基础。



## Products

具体的产品目录，每个产品拥有一个目录。

### Plugins

通用插件目录

### MySample

示例产品

#### UI

示例产品的UI

#### Plugins

示例产品的插件
