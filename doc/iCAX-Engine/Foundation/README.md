# Foundation

`Foundation` 记录 iCAX Engine 最底层基础能力的规格与方案文档。

这里的文档与源码目录 `src/icax/foundation/` 对齐，描述基础库的能力边界、公开 API 语义、内部实现方案和测试要求。Foundation 不依赖 `framework`、`runtime`、`services`、产品扩展或前端工程。

## 目录结构

- `ProcedureCall/`：过程调用封装的规格文档与方案文档。
- `Mailbox/`：普通 Mail 通信基础能力的规格文档和方案文档。
- `Task/`：C++ Task 基础能力的规格文档与方案文档。
