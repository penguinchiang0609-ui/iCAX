# 08 图层工艺与顺序规划详细设计

## 1. 模块定位

本模块负责刀路组织、MVP 工艺参数和加工顺序规划。

MVP 工艺仅包含：

- 微联。
- 引线。
- 引出。

激光功率、切割速度、气体、焦点、穿孔策略暂不进入当前模块。

## 2. 图层设计

```text
CLaserCamLayerComponent
  LayerID
  Name
  Color
  Visible
  LockState
  Order
```

规则：

- 每条刀路必须属于一个图层。
- 项目创建时自动创建默认图层。
- 图层隐藏只影响显示，不删除刀路。
- 图层锁定后不允许移动其中刀路或修改其工艺。

## 3. 工艺数据

```text
ToolpathProcess
  MicroJoint[]
  LeadIn
  LeadOut
```

```text
MicroJoint
  CurveParameter
  Length
```

```text
LeadInOut
  Type(Line/Arc)
  Length
  Radius
  Angle
```

曲线参数使用 `[0, 1]`。

## 4. 工艺修改流程

```text
1. 前端选择刀路
2. 用户修改微联/引线/引出
3. LaserCam.UpdateToolpathProcess
4. ModifyFilter 检查参数合法性
5. Repository 修改 ToolpathComponent
6. Observer 通知渲染和属性面板刷新
```

## 5. 工艺校验

微联：

- 参数必须在 `[0, 1]`。
- 长度必须大于 0。
- 多个微联不能重叠。

引线/引出：

- 类型必须是 Line 或 Arc。
- 长度必须大于 0。
- Arc 半径必须大于 0。

## 6. 顺序数据

```text
OrderPlan
  PlanID
  Version
  OrderedToolpath[]
  EstimatedAirMoveLength
```

```text
OrderedToolpath
  ToolpathID
  Direction(Forward/Reverse)
  LeadInParameter
```

## 7. 手动排序

支持：

- 上移。
- 下移。
- 拖拽。
- 手动正反向。
- 手动设置下刀点。

手动排序直接修改 OrderPlan。

## 8. 自动排序

MVP 策略：

```text
NearestNeighbor
```

考虑因素：

- 当前刀路结束点到下一刀路下刀点的距离。
- 下一刀路是否反向。
- 下刀点位置。
- 姿态变化成本。
- 普通空移和蛙跳成本。

评分：

```text
cost =
  airMoveDistanceWeight * distance
  + postureChangeWeight * postureDelta
  + liftPenalty
  + frogJumpPenalty
```

## 9. 与运动规划关系

顺序规划输出的是逻辑加工顺序。

运动规划会进一步生成：

- 切割段。
- 空移段。
- 蛙跳段。
- 姿态过渡。

顺序规划不直接生成关节运动。

## 10. 撤销还原

应支持撤销还原：

- 创建图层。
- 删除图层。
- 重命名图层。
- 修改图层颜色。
- 移动刀路到图层。
- 修改工艺。
- 修改顺序。
- 修改刀路方向。

## 11. 测试点

- 创建项目后存在默认图层。
- 刀路不能没有图层。
- 锁定图层后不能修改该图层刀路。
- 微联参数越界会被拒绝。
- 自动排序能输出完整刀路顺序。
- 手动拖拽后 OrderPlan 持久化。
