# 07 刀路曲线生成详细设计

## 1. 模块定位

刀路曲线生成模块负责把参数化特征转换为可持久化、可规划、可仿真的三维空间曲线。

刀路是 CAM 主数据，不能以 CAD 拓扑引用作为主身份。

## 2. 输入和输出

输入：

- RecognizedFeature。
- WorkpieceEntityID。
- 默认几何误差 `0.001 mm`。
- 可选用户生成参数。

输出：

- GeneratedToolpath。
- PathCurveResource。
- CCAMPathComponent。

## 3. 曲线类型

MVP 支持：

```text
LineSegment3
ArcSegment3
PolylineSegment3
EllipseSegment3
CubicBSplineSegment3
BezierSegment3
```

B 样条：

- MVP 支持三阶。
- 节点、控制点、权重必须可持久化。

## 4. 生成流程

```text
1. 读取 RecognizedFeature
2. 读取 Feature.Parameters 中的 path/depth/axis/direction 等参数
3. 必要时访问 BRep 或 SourceRefs 做精修
4. 生成世界坐标曲线段
5. 按曲线类型拟合
6. 检查闭合性
7. 计算最大偏差
8. 计算连续性等级
9. 输出 GeneratedToolpath
10. CommandHandler 写入 PathCurveResource
11. CommandHandler 创建 PathEntity
12. CommandHandler 将 PathEntity 追加到目标 Block.Children
13. 通知渲染同步
```

## 5. 误差控制

默认最大几何偏差：

```text
0.001 mm
```

计算方式：

- 对原始几何进行自适应采样。
- 对拟合曲线计算采样点距离。
- 对曲率变化大的区域增加采样密度。
- 最大距离超过默认误差时，拟合不合格。

## 6. 连续性等级

```text
PositionContinuous
TangentContinuous
CurvatureRadiusContinuous
CurvatureRadiusSmooth
```

规则：

- 最低必须位置连续。
- 同等误差下优先使用更高连续性。
- 尖角允许存在，但应在质量摘要中标记。

## 7. 闭合刀路

闭合 loop 生成闭合曲线。

闭合刀路必须支持：

- 下刀点参数。
- 正向/反向切割。
- 参数归一化。

```text
CurveParameter
  Range = [0, 1]
```

## 8. 质量摘要

```text
ToolpathQualitySummary
  AccuracyState(Passed/Failed/Warning)
  MaxDeviationInMillimeter
  ContinuityLevel
  MinCurvatureRadius
  CurvatureRadiusSmoothness
```

## 9. 数据写入

ToolpathGenerationService 不直接写 Repository。写入由 CommandHandler 在 undo step 中完成：

```text
ResourcePool.Add(PathCurveResource)
Repository.AddEntity(PathEntity)
Repository.AddComponent(CCAMPathComponent)
Repository.ModifyComponent(CCAMBlockComponent.Children)
```

ResourceID 必须先写入资源池，再写入 Database 引用。刀路生成后必须挂入某个 Block；默认挂入 `Root.ProgramRootBlockID` 指向的根 Block。

PathCurveResource 中的所有点、向量、圆弧平面、样条控制点和曲线参数都统一使用世界坐标。刀路曲线不保存 TCP、切割头局部坐标或前端渲染坐标。

## 10. 失败处理

- RecognizedFeature 缺少必要参数：失败。
- 原始几何不可读取：失败。
- 曲线拟合超过 `0.001 mm`：失败或标记不可确认。
- loop 不闭合：不能生成闭合刀路。
- 曲线自交：生成警告，是否允许由后续规则决定。

## 11. 测试点

- line edge 生成 line segment。
- arc edge 生成 arc segment。
- 复杂 loop 可拟合三阶 B 样条。
- 最大偏差超过 `0.001 mm` 时不能静默通过。
- 保存重开后刀路曲线无需 CAD 拓扑即可恢复。
