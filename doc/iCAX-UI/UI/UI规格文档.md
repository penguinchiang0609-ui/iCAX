# UI 规格文档

## 1. 定位

`UI` 是公共 UI 基础模块。

它提供跨 AppShell 和产品 webpage 可复用的 UI 工具与组件。

## 2. 当前能力

当前提供：

- HTML 文本转义。
- HTML 属性转义。

公共组件范围：

- 公共控件。
- 布局组件。
- 主题 token。
- 图标封装。

## 3. 边界

本模块不包含业务状态，不调用 Facade，不访问 PDO。

## 4. HTML 安全规则

所有通过模板字符串插入 DOM 的动态文本必须先转义。

文本节点使用：

```js
escapeText(value)
```

属性值使用：

```js
escapeAttr(value)
```

产品模块如果使用模板字符串渲染，也应复用本模块工具。

## 5. 公共组件规则

公共组件必须满足：

- 不绑定具体产品业务。
- 不直接访问 backend。
- 不直接访问 PDO。
- 可以被 AppShell 和产品 webpage 复用。

适合放入本模块的内容：

- 按钮、工具条、面板、列表等基础控件。
- 主题 token。
- 图标封装。
- 空状态和错误状态组件。

不适合放入本模块的内容：

- 平切 CAM 特有面板。
- 五轴刀路编辑器。
- 某个产品的参数表单。

## 6. 验收要求

- 动态文本插入 DOM 前有转义。
- 公共 UI 不依赖 Facade/PDO/runtime。
- 产品专用 UI 不进入公共模块。
